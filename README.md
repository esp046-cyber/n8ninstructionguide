# n8ninstructionguide
Features:
n8n-inspired design: Red/orange gradient theme (#ff6d5a) matching n8n's brand colors
Dark mode interface: Professional dark theme with good contrast
Mobile-optimized: 1% margins, responsive layout, touch-friendly
Small font titles: Compact headers as requested
5 difficulty levels: From "Pinakamahirap" (Hardest) to "Pinakamadali" (Easiest)
25 tasks total: Covering everything from OAuth2 integration to basic navigation
Collapsible sections: Tap headers to expand/collapse content
Step-by-step instructions: Clear, numbered steps in Filipino
Code examples: For advanced tasks
Tips: Helpful hints throughout
Tasks organized by difficulty:
Hardest (1-5): OAuth2, custom code, databases, error handling
Hard (6-10): Conditionals, loops, scheduling, multi-API
Medium (11-15): HTTP requests, webhooks, filtering, emails
Easy (16-20): First workflow, triggers, activating workflows
Easiest (21-25): Interface navigation, connecting nodes, renaming
